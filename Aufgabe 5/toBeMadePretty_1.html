Hier ein paar Gründe, warum CSS nicht zu unterschätzen ist:

  1. People love pretty things!
  2. Wie diese Seite zeigt ʕ•ᴥ•ʔ, kann CSS die Übersichtlichkeit einer Website verbessern.
  3. CSS ist einfach wiederzuverwenden, indem man einfach Klassen-Namen und ggfs. IDs wiederverwendet.
  4. Zu große Bilder können leicht verkleinert, und sogar neben Text positioniert werden.
  5. User experience is Key!
